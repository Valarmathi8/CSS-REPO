HTML Terms

HTML
HTML is the standard markup language for Web pages
HTML element
An HTML element is defined by a start tag, some content, and an end tag.
HTML tag
The <html> tag represents the root of an HTML document.
Angle bracket
Angle brackets, sometimes known as chevrons, are a pair of punctuation marks that take the form of<>
HTML attribute
HTML attributes provide additional information about HTML elements.
HTML comment
The comment tag is used to insert comments in the source code.
HTML object element
The object tag defines a container for an external resource.
HTML special character
To add such symbols to an HTML page, you can use the entity name or the entity number for the symbol
